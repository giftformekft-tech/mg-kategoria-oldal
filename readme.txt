== Description ==
Adds a category switcher (chips/buttons) to WooCommerce product category archive pages.

Admin UI:
WooCommerce -> Kategória váltó
- Display mode: scroll (single row on mobile) or wrap (multi-row show all)
- Hide empty categories
- Show/hide product counts

== Installation ==
1. Upload the ZIP in WordPress: Plugins → Add New → Upload Plugin.
2. Activate.
3. Configure: WooCommerce → Kategória váltó
4. Visit any product category page.
